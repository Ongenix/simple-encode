# simple-encode

## Encodes & decodes text

# Usage
## Simple usage
python3 main.py

# Console

## To encode:
python3 main.py encode "hi" --console
## To decode:
python3 main.py decode "ij" --console

### You can also pipe to a file:
## To encode:
python3 main.py encode "hi" encoded.txt
## To decode:
python3 main.py decode "ij" decoded.txt
